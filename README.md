○Tips
------
* 箇条書きの入れ子は 4 スペースなので注意。

```
* 箇条書き
    * 入れ子の箇条書き
```

* 最近は 2 スペースが多そう？GFM も 2 スペース

```
* 箇条書き
  * 入れ子の箇条書き
```

* 箇条書きの改行は後ろにスペース 2

```
* 箇条書き 1 行目__ ← 2 スペース
  2 行目
```
